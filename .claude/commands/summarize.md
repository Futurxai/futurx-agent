# /summarize command
When the user types /summarize:
1. Ask: "Paste your meeting transcript or voice note summary here"
2. Wait for input
3. Extract and output:
   - Key decisions (what was agreed)
   - Action items (task | owner | deadline)
   - Blockers (what is stopping progress)
   - Next steps (what happens this week)
4. Format as a clean table
5. Ask: "Should I update ROADMAP.md checkboxes based on this?"
