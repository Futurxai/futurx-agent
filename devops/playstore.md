# Play Store Release Guide

## Apps to Release
1. EatWell
2. Paydll
3. LoveDigitally
4. PGSeat

## Pre-Release Checklist
- [ ] Signed release APK/AAB generated
- [ ] App icons (512x512 hi-res, 48x48 launcher)
- [ ] Feature graphic (1024x500)
- [ ] Screenshots (phone + tablet, min 2 each)
- [ ] Privacy policy URL hosted
- [ ] Content rating questionnaire completed
- [ ] Target API level meets Play Store requirements

## Steps
1. Create Google Play Console developer account
2. Create app listing per app
3. Upload AAB (not APK) for production
4. Fill store listing: title, short desc, full desc
5. Set pricing (free) and distribution (India first)
6. Submit for review
