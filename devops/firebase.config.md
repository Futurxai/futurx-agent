## Services Used
- Firebase Authentication (Google Sign-In)
- Firebase Analytics
- Firebase Crashlytics
- Firebase Cloud Messaging (FCM)
- Firebase Remote Config
- Firebase Performance Monitoring
- Firebase Hosting

## Setup Steps
1. Create Firebase project per app
2. Add Android app with package name
3. Download google-services.json → add to android/app/
4. Enable Analytics, Crashlytics, FCM in console
5. Never commit google-services.json to public repos
