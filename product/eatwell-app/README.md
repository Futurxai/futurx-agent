# EatWell4U - Personalized Meal Planning App

## Overview
A full-featured meal planning mobile app converted from a single HTML prototype to a production-ready Ionic Angular app with Android APK and Firebase hosting.

## Live
- **Web:** https://eatwell-5eaf0.web.app
- **Repo:** https://github.com/Futurxai/EatWell4U

## Features
- Google Sign-In (Firebase Auth)
- 6-step profile setup (physical, diet, cuisine, food preferences)
- Dashboard with daily meals & calorie tracking
- Meal swap & recipe browser (50+ recipes, 5 cuisines)
- Weekly 7-day meal planner with PDF export
- Grocery list with category filters & progress tracking
- Dark mode toggle
- Meal reminder push notifications
- Camera food scanner with swap suggestions
- Signed Android APK (7.2 MB)

## Tech Stack
Ionic 7 | Angular 17 | Capacitor 7 | Firebase Auth | Firebase Hosting | TypeScript | SCSS

## Cuisines Supported
South Indian | North Indian | Continental | Chinese | Healthy/Default
