## Marketing Thinking Framework

### Always Answer These First
```
WHO:    Who is the exact target audience? (Be specific — not "everyone")
WHAT:   What is the ONE key message? (Not three. One.)
WHY:    Why should they care? (User benefit, not feature)
WHERE:  What channel? (Each channel needs different content)
WHEN:   Where in the funnel? (Awareness / Consideration / Decision)
```

### The Copywriting Formula (Universal)
```
Problem → Agitate → Solve (PAS)
  OR
Attention → Interest → Desire → Action (AIDA)
  OR
Feature → Advantage → Benefit (FAB)
```

## Quick Copy Templates

### Value Proposition (1 sentence)
```
We help [target user] [achieve outcome] without [pain/sacrifice].
```

### Hero Headline Options
```
Action:     "Ship 10x faster with AI code review"
Question:   "Tired of losing customers to slow support?"
How-to:     "How [Company] reduced churn by 40% in 30 days"
Number:     "Join 50,000 developers who ship without bugs"
Outcome:    "The only tool you need to go from idea to launch"
```

### CTA Hierarchy
```
Primary:    "Start free trial" / "Get started free" / "Try it now"
Secondary:  "See how it works" / "Watch demo" / "Learn more"
Soft:       "Explore features" / "Read the docs"
```

## Output Standards
- Lead with benefit, not feature
- Use active voice, short sentences
- Specifics beat vague claims ("40% faster" not "much faster")
- One CTA per piece — don't confuse the reader
- Match tone to platform (LinkedIn ≠ Twitter ≠ Email)
