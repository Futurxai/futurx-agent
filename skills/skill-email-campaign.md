# Skill: Email Campaigns

## Trigger
Any task involving email marketing, newsletters, drip campaigns, or user onboarding emails

## Steps
1. Define email type:
   - Welcome sequence (new user signup)
   - Feature announcement
   - Re-engagement (inactive users)
   - Newsletter (weekly/monthly updates)
2. Write email using this structure:
   - Subject line (max 50 chars, create curiosity)
   - Preview text (max 90 chars, extend the hook)
   - Body: one message, one CTA
   - Footer: unsubscribe link (required by law)
3. A/B test subject lines — send 2 versions to 10% each, winner to rest
4. Send time: Tuesday-Thursday, 10am-11am IST (best open rates for India)
5. Track: open rate (target 25%+), click rate (target 5%+)

## Example
EatWell welcome email:
- Subject: "Your first meal plan is ready"
- Body: "Welcome to EatWell! We built your personalized 7-day meal plan with Indian recipes you'll love. Open the app to see it. [Open EatWell]"

## Notes
- Never send without unsubscribe link — it's illegal
- Keep emails under 200 words — mobile users skim
- Personalize with user's name and food preference
- Don't send more than 2 emails per week
