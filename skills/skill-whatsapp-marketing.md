# Skill: WhatsApp Marketing

## Trigger
Any task involving WhatsApp campaigns, broadcast messages, or WhatsApp Business API

## Steps
1. Define campaign goal: launch / update / feedback / re-engagement
2. Define audience segment: new users / active users / churned users
3. Write message template (max 1024 chars) with:
   - Greeting (personalized with name if possible)
   - One clear message (what + why)
   - One CTA button (link or reply keyword)
4. Translate to Tamil and English — both versions ready
5. Schedule send time: 10am-12pm or 6pm-8pm IST (best engagement)
6. Set up tracking: link UTM params for click tracking
7. Follow up: send reminder after 48 hours to non-openers

## Example
EatWell launch message:
"Hi {{name}}! EatWell is live — plan your meals with 50+ Indian recipes. Download free: [link]. Reply STOP to unsubscribe."

## Notes
- Max 2 messages per week per user — no spam
- Every message must have opt-out option
- WhatsApp Business API required for bulk sends (not regular WhatsApp)
- Test with 10 people before sending to full list
- Tamil message should feel natural, not translated
