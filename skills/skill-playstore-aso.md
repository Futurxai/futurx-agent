# Skill: Play Store ASO (App Store Optimization)

## Trigger
Any task involving Play Store listing, app description, keywords, or app visibility

## Steps
1. Research top 5 competitor apps in the same category
2. Write app title (max 30 chars) — include main keyword
3. Write short description (max 80 chars) — one-line value proposition
4. Write full description (max 4000 chars) — use this structure:
   - Hook (first 2 lines visible before "Read more")
   - Key features (bullet points)
   - How it works (3 steps)
   - Social proof / stats
   - Call to action
5. List 5-10 keywords for search ranking
6. Prepare screenshot captions (what each screen shows)
7. Write "What's New" section for updates

## Example
EatWell Play Store listing:
- Title: "EatWell — Indian Meal Planner"
- Short: "Track meals, count calories with 50+ Indian recipes"
- Keywords: meal planner india, calorie tracker tamil, indian food diet

## Notes
- Always write in both Tamil and English
- First 2 lines of description are most important — users see only that
- Update keywords every month based on search trends
- Screenshots must show real app UI, not mockups
