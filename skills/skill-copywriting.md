# Skill: Copywriting

## Trigger
Any task involving ad copy, headlines, taglines, app descriptions, landing page text, or marketing copy

## Steps
1. Identify the audience: who is reading this?
2. Identify the goal: download / signup / click / buy
3. Use one of these formulas:
   - **PAS**: Problem → Agitate → Solve
   - **AIDA**: Attention → Interest → Desire → Action
   - **BAB**: Before → After → Bridge
4. Write 3 headline options — pick the best
5. Rules:
   - Lead with benefit, not feature
   - Use numbers ("50+ recipes" not "many recipes")
   - Active voice ("Track your meals" not "Meals can be tracked")
   - One CTA per piece
   - Max 2 sentences per paragraph
6. Write in both Tamil and English

## Example
EatWell ad copy:
- Feature: "50+ Indian recipes"
- Benefit: "Eat healthy without giving up your favorite Indian food"
- CTA: "Download EatWell free"

## Notes
- Test 3 versions of every headline — data beats opinion
- Short copy for ads (under 30 words), long copy for landing pages
- Always match tone to platform (WhatsApp = casual, LinkedIn = professional)
- Tamil copy should sound natural, not translated from English
