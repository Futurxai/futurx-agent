# Skill: Social Media Marketing

## Trigger
Any task involving Instagram, Twitter/X, LinkedIn, YouTube, or social posts

## Steps
1. Identify platform: each platform has different rules
   - Instagram: visual, reels, stories, carousel
   - Twitter/X: short, threads, engagement
   - LinkedIn: professional, long-form, company updates
   - YouTube: tutorials, demos, shorts
2. Write post using PAS formula: Problem → Agitate → Solve
3. Add hashtags (5-10 relevant, mix of popular + niche)
4. Create content calendar: 3-5 posts per week
5. Best posting times for India:
   - Instagram: 12pm-2pm, 7pm-9pm IST
   - Twitter: 9am-11am, 6pm-8pm IST
   - LinkedIn: 8am-10am IST (weekdays)
6. Track engagement: likes, shares, comments, link clicks

## Example
Instagram post for EatWell:
"Tired of counting calories for Indian food? EatWell has 50+ local recipes with exact nutrition data. Idli to Dal Makhani — we got you. Download link in bio. #EatWell #IndianFood #MealPlanner #HealthyEating #TamilNadu"

## Notes
- Tamil content performs better in Tamil Nadu market
- Video/reels get 3x more engagement than static posts
- Respond to every comment within 24 hours
- Never buy followers — ruins algorithm reach
