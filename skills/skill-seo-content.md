# Skill: SEO & Content Writing

## Trigger
Any task involving blog posts, landing page copy, website content, or search ranking

## Steps
1. Research keywords using Google Trends (India-specific)
2. Pick 1 primary keyword + 3-5 secondary keywords
3. Write content structure:
   - H1: Include primary keyword (max 60 chars)
   - Meta description (max 155 chars) — include keyword + CTA
   - Introduction: hook + what reader will learn
   - H2 sections: one per subtopic, include secondary keywords
   - Conclusion: summary + CTA
4. Content rules:
   - Min 800 words for blog posts
   - Short paragraphs (2-3 sentences max)
   - Use bullet points and numbered lists
   - Add internal links to other pages
   - Add one image per 300 words
5. Optimize for futurx.in pages

## Example
Blog post: "Best Meal Planning Apps in India 2026"
- Primary keyword: "meal planning app india"
- H2s: "Why Meal Planning?", "Top 5 Apps", "EatWell Features", "How to Start"

## Notes
- Write for humans first, SEO second
- Tamil content ranks well for local searches — create Tamil versions
- Update content every 3 months to stay fresh
- Add schema markup for rich snippets
